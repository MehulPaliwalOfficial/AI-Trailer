# Submission Checklist

Use this checklist before submitting the project.

## Required Files

- [x] Source code folder: `src/`
- [x] Prompt document: `prompts/prompt_document.md`
- [x] Project report: `docs/project_report.md`
- [x] Architecture note: `docs/architecture.md`
- [x] Requirements file: `requirements.txt`
- [x] Sample story output
- [x] Sample trailer script output
- [x] Sample narration output
- [x] Sample image prompts
- [x] Sample voice-over MP3
- [x] Sample final trailer MP4

## Personal Details to Fill

Update these in the report/PPT if needed:

- Student name
- Roll number
- College name
- Department
- Course / subject
- Faculty name

## Demo Steps

1. Open the project folder.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run src/app.py`.
4. Enter movie title, genre, and optional idea/plot details.
5. Click **Generate Trailer**.
6. Show generated story, script, narration, image prompts, and final trailer.

## Recommended Submission Package

Submit:

- `AI_Movie_Trailer_Generator_Submission.zip`
- Existing presentation file if separately required
- Final video from `sample_outputs/final_movie_trailer.mp4` if the portal asks for video separately
