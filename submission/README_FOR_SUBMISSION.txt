AI Movie Trailer Generator - Submission Notes

This folder is a complete project package. It contains:
- Python source code
- Streamlit app
- Prompt document
- Project report
- Architecture notes
- Sample generated outputs
- Final sample trailer video

Main video file:
sample_outputs/final_movie_trailer.mp4

How to run:
1. pip install -r requirements.txt
2. streamlit run src/app.py

Optional Gemini API:
Set GEMINI_API_KEY in your environment or in a .env file.
If no API key is set, the app runs in offline demo mode.

Before final submission, fill in your name, roll number, college, and department in docs/project_report.md and in the PPT if required.
